How It Works

This app uses a machine learning model trained on real-world credit card transaction data to detect fraud.

  1. Step 1: Scroll down to the Demo: Transaction Input section.
  2. Step 2: Fill in the required fields:
    • Time: Time of the transaction in seconds since the first recorded transaction.
    • V1 to V28: These are anonymized features derived from PCA (you can input test values).
    • Amount: The transaction amount in USD or your currency.
  3. Step 3: Click the "Check Transaction" button.
  4. Step 4: The system will instantly tell you whether the transaction is Safe or Fraudulent.

You can test it with your own data or use sample values for demonstration. This is a demo tool and not intended for production financial use.
